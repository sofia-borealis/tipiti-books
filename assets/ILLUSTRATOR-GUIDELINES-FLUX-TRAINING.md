# Lineamientos para Ilustradora — Tipiti Books
## Imágenes de Referencia para Entrenamiento de Modelo FLUX Kontext Max

> **Objetivo:** Este documento es la guía para la ilustradora contratada. Las imágenes que cree servirán como referencia visual para entrenar un modelo de IA (FLUX Kontext Max) que luego generará todas las variantes del personaje automáticamente. La calidad y consistencia de estas imágenes de referencia es CRÍTICA — todo lo que genere el modelo después dependerá de ellas.

---

## 1. Resumen del Proyecto

**Tipiti Books** es una marca chilena de libros infantiles personalizados. Cada libro tiene un personaje principal (niño o niña) que el comprador configura para que se parezca al destinatario del regalo. El estilo visual es **acuarela pintada a mano**, cálido y tierno, con calidad de libro infantil editorial premium.

**Tu rol:** Crear las imágenes de referencia "maestras" que definirán el estilo visual de toda la marca. Estas imágenes se usarán para entrenar un modelo de IA que generará variantes del personaje (distintos tonos de piel, colores de pelo, etc.) manteniendo tu estilo artístico.

**Por qué importa:** Si tus imágenes son consistentes y de alta calidad, el modelo aprenderá tu estilo y lo replicará fielmente en las ~80 variantes de cada libro. Si hay inconsistencias, el modelo las amplificará.

## 2. Estilo Artístico — La Biblia Visual

### LO QUE QUEREMOS

| Elemento | Descripción |
|----------|-------------|
| **Técnica** | Acuarela sobre papel — con pinceladas visibles, bordes suaves, manchas de color naturales |
| **Paleta** | Pasteles cálidos: rosa viejo, amarillo suave, verde salvia, azul empolvado, terracota suave. Base crema/off-white (como papel acuarela real) |
| **Textura** | Visible — se debe notar la fibra del papel y las capas de acuarela. Nada liso ni digital |
| **Iluminación** | Cálida, como luz de lámpara de noche. Dorados suaves, sombras delicadas |
| **Emoción** | Tierno, acogedor, como un abrazo. Que transmita la sensación de "hora de dormir" |
| **Nivel de detalle** | Rico en pequeños detalles encantadores (peluches, animalitos, objetos curiosos) pero sin saturar |
| **Calidad editorial** | Nivel de libro infantil publicado por una editorial premium (no "sketch" ni "boceto") |

### LO QUE NO QUEREMOS

| Evitar | Por qué |
|--------|---------|
| ❌ 3D / Pixar / Disney CGI | Queremos acuarela artesanal, no animación digital |
| ❌ Anime / Manga | No es el estilo de la marca |
| ❌ Clipart / Vectorial plano | Necesitamos textura y profundidad |
| ❌ Fotorrealismo | Es un libro infantil, no una foto |
| ❌ Colores primarios saturados | Nada de rojo puro, azul eléctrico, amarillo chillón |
| ❌ Líneas negras gruesas (estilo cómic) | Queremos bordes suaves y difuminados, propios de acuarela |
| ❌ Fondo blanco puro | Siempre fondo crema/papel acuarela con textura |
| ❌ Texto, letras o palabras dentro de la imagen | NUNCA — el texto se agrega después digitalmente |

### Artistas de Referencia

Para entender la dirección visual, estudia estos referentes:
- **Oliver Jeffers** — *Lost and Found*, *Here We Are*: acuarela emocional, suelta, con personalidad
- **Benji Davies** — *The Storm Whale*: acuarela con atmósferas cálidas y profundidad
- **Jon Klassen** — *I Want My Hat Back*: simplicidad expresiva, ojos como puntos
- **Isabelle Arsenault** — *Colette's Lost Pet*: paleta pastel, textura de papel real

**Nota:** No queremos copiar a ninguno — queremos un estilo propio para Tipiti Books que viva en ese mismo universo visual.

## 3. El Personaje — Especificaciones Técnicas

### 3.1 Anatomía Base

El personaje es un **toddler** (niño/a de ~2-3 años) con las siguientes características FIJAS:

| Característica | Especificación |
|----------------|----------------|
| **Edad visual** | 2-3 años (toddler, proporciones de bebé grande) |
| **Proporciones** | Cabeza grande respecto al cuerpo (~1/3 de la altura total), extremidades cortas y redondeadas |
| **Ojos** | Dos puntitos negros redondos. SIN iris, SIN color, SIN brillo. Solo dos puntos negros sólidos |
| **Nariz** | Apenas sugerida — un puntito o pequeña curva |
| **Boca** | Pequeña, simple, expresiva. Puede ser una línea curva, una "u", o una "o" pequeña |
| **Mejillas** | Ligeramente rosadas (rubor de acuarela) |
| **Manos** | Simplificadas, redondeadas, sin dedos muy definidos |
| **Vestimenta** | Pijama cómodo y suelto (el primer libro es "Buenas Noches") |

### 3.2 Variantes del Personaje

El comprador configura su personaje eligiendo entre estas opciones. Necesitamos que ilustres la **variante base** (que se indica más abajo) y unas pocas variantes adicionales para que el modelo aprenda las diferencias:

| Atributo | Opciones |
|----------|----------|
| **Género** | Niño (pelo corto), Niña (pelo largo) |
| **Tono de piel** | Claro (pálido/rosado), Mate (oliva/medio), Oscuro (marrón profundo cálido) |
| **Color de pelo** | Castaño, Pelirrojo, Negro. *Para piel clara y mate también: Rubio* |
| **Tipo de pelo** | Liso, Ondulado |
| **Lentes** | Sí (redondos pequeños), No |

**Nota biológica importante:** Para piel oscura, solo se ofrece pelo negro (no rubio ni pelirrojo). Para piel clara y mate se ofrecen los 4 colores.

### 3.3 Variante Base para Ilustrar

**Empezar con esta variante como referencia principal:**
- Niña, piel mate (oliva), pelo castaño ondulado, sin lentes
- Pijama celeste claro con estrellitas blancas

**Variantes adicionales mínimas a ilustrar** (para que el modelo aprenda las diferencias):
1. Niño, piel clara, pelo rubio liso, sin lentes — pijama verde suave
2. Niña, piel oscura, pelo negro ondulado, sin lentes — pijama rosa viejo
3. Niño, piel mate, pelo negro liso, con lentes redondos — pijama amarillo suave
4. Niña, piel clara, pelo pelirrojo liso, con lentes redondos — pijama lavanda

## 4. Entregables — Qué Necesitamos que Ilustres

### 4.1 PORTRAIT (Money Shot) — 1 por variante

**Qué es:** Un retrato frontal del personaje que define su identidad visual PARA SIEMPRE. Es la imagen "maestra" de la que se derivará todo lo demás.

**Especificaciones:**
- Vista: Frontal (mirando al espectador)
- Encuadre: Desde el pecho hacia arriba
- Expresión: Cálida, amigable, levemente soñolienta (es hora de dormir)
- Fondo: Neutro, crema cálido con textura de acuarela (sin elementos de escena)
- Resolución: Mínimo 2048 × 2048 px
- Formato: PNG con fondo (NO transparente)

**Cantidad:** 5 portraits (1 variante base + 4 variantes adicionales listadas arriba)

### 4.2 CHARACTER SHEET — 1 por variante

**Qué es:** Una lámina con 4 vistas del mismo personaje, para que el modelo entienda cómo se ve desde distintos ángulos.

**Las 4 vistas:**
1. **Frontal** — mirando de frente (puede ser igual al portrait pero de cuerpo entero)
2. **Lateral** — perfil (mirando a la derecha)
3. **Tres cuartos (¾)** — girado ~45° respecto al frontal
4. **Cuerpo completo** — de pie, mostrando proporciones y vestimenta completa

**Especificaciones:**
- Las 4 vistas deben estar en una SOLA imagen (como una "model sheet" de animación)
- Fondo neutro crema para todas las vistas
- El personaje debe verse IDÉNTICO en las 4 vistas (mismas proporciones, colores, detalles)
- Resolución: Mínimo 4096 × 2048 px (horizontal, 4 vistas lado a lado)
- Formato: PNG

**Cantidad:** 5 character sheets (1 por variante)

### 4.3 SETTING SHEET — 1 por libro (no por variante)

**Qué es:** Una lámina con 3 vistas del ambiente/escenario donde transcurre la historia. Para el primer libro ("Buenas Noches") es la habitación del niño/a.

**Las 3 vistas:**
1. **Frontal** — vista principal de la habitación (desde la puerta)
2. **Lateral** — misma habitación desde el costado de la cama
3. **Detalle** — close-up de un rincón con elementos recurrentes (mesita de noche, peluche, etc.)

**Especificaciones:**
- SIN personaje en la escena (la habitación vacía, lista para que el modelo coloque al personaje)
- Elementos que DEBEN estar: cama con almohadas y cobija suave, ventana con estrellas/luna, lámpara de noche encendida (luz cálida), peluches, libros apilados, un ratoncito dormido en algún rincón (easter egg recurrente)
- Misma paleta pastel cálida del personaje
- Resolución: Mínimo 4096 × 2048 px
- Formato: PNG

**Cantidad:** 1 setting sheet para "Buenas Noches"

### 4.4 ESCENAS DE REFERENCIA — 3 escenas completas

**Qué es:** 3 escenas terminadas (personaje + ambiente) que muestren cómo se ve una página final del libro. Esto le enseña al modelo cómo integrar personaje + setting en una composición editorial.

**Especificaciones de composición para cada escena:**
- Formato horizontal (ratio ~22:18, similar al libro impreso)
- El personaje debe estar a la IZQUIERDA o DERECHA, nunca en el centro exacto
- Dejar un espacio vacío (sin ilustración) donde irá el texto — aproximadamente 25% de la imagen
- Incluir al menos un "easter egg": un animalito escondido, un peluche en un lugar inesperado, una mariposa, etc.

**Las 3 escenas sugeridas (usando la variante base):**
1. El personaje sentado en la cama con un peluche, bostezando. Luz cálida de lámpara.
2. El personaje mirando por la ventana las estrellas, de espaldas/perfil. Luz de luna azulada.
3. El personaje dormido, acurrucado con una cobija suave. Iluminación tenue, sensación de paz.

**Resolución:** Mínimo 5197 × 2126 px (300 DPI para impresión 22×18 cm)
**Formato:** PNG

**Cantidad:** 3 escenas

### 4.5 STYLE REFERENCES — 3 a 5 imágenes adicionales

**Qué es:** Imágenes que muestren tu estilo artístico sin personaje ni escena específica. Pueden ser:
- Texturas de acuarela sobre papel
- Objetos sueltos pintados en tu estilo (un peluche, una taza de leche, la luna, estrellas, flores)
- Patrones decorativos que podrían usarse en guardas del libro
- Detalles de tus pinceladas características

**Propósito:** Ayudan al modelo a entender tu técnica de acuarela, textura y paleta más allá de los personajes.

**Resolución:** Mínimo 2048 × 2048 px
**Formato:** PNG
**Cantidad:** 3-5 imágenes

## 6. Reglas Inquebrantables

Estas reglas deben respetarse en TODAS las imágenes sin excepción:

1. **NUNCA incluir texto, letras, palabras o números dentro de la imagen.** El texto se agrega después digitalmente. Si estás tentada de escribir el nombre del niño o "Buenas Noches" — NO lo hagas.

2. **Los ojos siempre son dos puntitos negros.** Sin iris, sin color, sin brillo, sin pupilas detalladas. Dos puntos negros redondos y nada más.

3. **Consistencia entre vistas.** El mismo personaje debe tener exactamente las mismas proporciones, colores, y detalles en el portrait, el character sheet, y las escenas. Si el pijama tiene estrellitas blancas en el portrait, las tiene en todas las demás imágenes.

4. **Fondo siempre con textura.** Nunca blanco puro. Siempre crema/off-white con textura visible de papel acuarela.

5. **Sin bordes ni marcos.** Las imágenes llegan hasta el borde (bleed). No agregar marcos, bordes, líneas divisorias, ni decoraciones de "hoja" alrededor.

6. **La imagen debe ser 100% tuya.** No usar imágenes de stock, assets de terceros, ni generaciones de IA como base. Necesitamos ilustraciones originales para entrenar el modelo legalmente.

## 7. Proceso de Trabajo Sugerido

1. **Fase 1 — Variante base:** Crear el portrait y character sheet de la variante base (niña mate, pelo castaño ondulado). Enviar para revisión antes de continuar.
2. **Fase 2 — Setting:** Crear el setting sheet de "Buenas Noches". Enviar para revisión.
3. **Fase 3 — Escenas:** Crear las 3 escenas completas con la variante base. Enviar para revisión.
4. **Fase 4 — Variantes adicionales:** Crear portraits y character sheets de las 4 variantes adicionales.
5. **Fase 5 — Style references:** Crear las 3-5 imágenes de estilo/textura.

**Revisiones:** Incluimos 2 rondas de revisión por fase. Los ajustes típicos serán de proporciones, paleta de colores, y nivel de detalle.

## 8. Especificaciones Técnicas para Entrega

- **Formato de archivo:** PNG (sin compresión lossy)
- **Espacio de color:** sRGB para las imágenes digitales. Si puedes entregar también versión CMYK, ideal (para impresión)
- **Resolución mínima:** 300 DPI en tamaño de impresión (22 × 18 cm para escenas)
- **Naming convention:**
  - `portrait-[variante].png` → ej: `portrait-girl-medium-brown-wavy.png`
  - `charsheet-[variante].png` → ej: `charsheet-boy-light-blonde-straight.png`
  - `setting-buenas-noches-[vista].png` → ej: `setting-buenas-noches-all.png`
  - `scene-buenas-noches-[número].png` → ej: `scene-buenas-noches-01.png`
  - `style-ref-[número].png` → ej: `style-ref-01.png`

## 9. Contexto sobre el Entrenamiento del Modelo

Para que entiendas cómo se usarán tus imágenes:

1. Tus **portraits y character sheets** se usarán para entrenar al modelo FLUX Kontext Max a "entender" tu estilo de personaje. Luego, el modelo generará ~80 variantes (distintas combinaciones de piel, pelo, lentes) replicando tu estilo.

2. Tu **setting sheet** se usará como referencia de ambiente. El modelo lo usa para mantener el escenario consistente en todas las páginas.

3. Tus **escenas completas** le enseñan al modelo cómo componer una página final: dónde va el personaje, cómo interactúa con el ambiente, dónde queda el espacio para texto.

4. Tus **style references** le enseñan la textura y técnica de acuarela para que no derive hacia estilos digitales.

**Cuanto más consistente y de alta calidad sea tu trabajo, mejor será el resultado del modelo.** Por eso las reglas de consistencia son tan estrictas — no es perfeccionismo, es necesidad técnica.
